405 탐색
    - 주어진 데이터에서 자신이 원하는 데이터를 찾아내는 알고리즘



05-1 깊이 우선 탐색(Depth-First Search)
    - 그래프 완전 탐색 기법
    - 그래프의 시작 노드에서 출발, 탐색할 한 쪽 분기를 정하여 최대 깊이까지 탐색을 마친 후 다른 분기로 이동하여 다시 탐색을 수행
    - 재귀 함수로 구현
    - 스택 자료구조 이용
    - 시간복잡도 O(V+E) # V: 노드 수, E: 엣지 수
    - 스택 오버플로에 유의해야 한다

    - 깊이 우선 탐색 응용 문제
        - 단절점 찾기
        - 단절선 찾기
        - 사이클 찾기
        - 위상 정렬

    - 핵심이론:
        - 한 번 방문한 노드는 다시 방문하지 않음
            - 방문여부 체크리스트 필요
        - 그래프는 인접 리스트로 표현
        - 후입선출, 스택 사용(스택 안하고 재귀함수해도 무관)

05-2 너비 우선 탐색(Breadth-First Search)
    - 그래프 완전 탐색
    - 시작 노드에서 출발해 시작 노드를 기준으로 가장 가까운 노드를 먼저 방문하면서 탐색하는 알고리즘
    - FIFO, Queue 이용
    - 시간 복잡도 O(V+E) # V: 노드 수, E: 엣지 수
    - 경로가 여러가지 일 때 최단 경로를 보장한다.

    - 핵심이론
        1. 시작 노드를 정하고 사용할 자료구조를 초기화
            - 방문 했던 노드를 체크하는 리스트 필요
            - 그래프를 인접 리스트로 표현
            - 스택 아닌 큐 사용
        2. 큐에서 노드를 꺼낸 후 노드의 인접 노드를 다시 큐에 삽입
            - 큐에서 노드를 꺼낸 후 노드의 인접 노드를 다시 큐에 삽입
            - 방문리스트 체크, 방문한 노드는 큐에 삽입하지 않음
            - 큐에서 꺼낸 노드는 탐색 순서에 기록
        3. 큐 자료구조에 값이 없을 때까지 반복
            - 큐에 노드가 없을 때까지 1,2 과정을 반복.
            - 선입선출 방식이므로 탐색순서가 DEF와 다르다
